### Installation 

# GIT

```shell
wget https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPO/releases/latest/download/sekrets-linux.tar.gz

tar -xzvf sekrets-linux.tar.gz
cd sekrets-release
sudo make install
```
